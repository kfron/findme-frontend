Angular CLI do projektu
1. Stworzy plik angular.json
2. Dodac zawartosc:
	{
	    "version": 1,
	    "cli": {
		"defaultCollection": "@nativescript/schematics"
	    },
	    "projects": {
		"nazwa": {
		    "root": "",
		    "sourceRoot": "src",
		    "projectType": "application",
		    "prefix": "ns"
		}
	    },
	    "defaultProject": "nazwa"
	}
	
	
Stworzenie projektu
1. ns create nazwa --angular
2. ns platform add android

Uzyty emulator:
Nexus 4 API 28 Software Graphics Cold Boot


Uruchamianie projektu:
Frontend -
	/Projekt/findme/ $ ns run android
Backend -
	/Projekt/backend/ $ npm start
Backend Debug -
	/Projekt/backend/ $ DEBUG=backend:* npm start
	
Connect to db from terminal
	psql -d findme
